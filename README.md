# Project RT

**Project RT** is a minimal web-based reaction time test.

Wait on the red screen,  
tap or click as soon as the screen turns green,  
and your reaction time will be measured in milliseconds.

---

## Features

- 🎯 Reaction time test with random delay
- 🔁 Measures 5 attempts per session
- 📊 Displays the average result with a visual chart
- 📱 Works on both mobile and desktop
- ⚡ Single-file, lightweight implementation

---

## Haptic Feedback (Mobile)

- **Android devices** support haptic feedback (vibration) on success and failure.
- **iOS (iPhone)** does **not** support web vibration due to browser and OS restrictions.

> This is expected behavior based on platform policies, not a bug.

---

## How to Use

1. Tap / click the screen to start
2. Wait while the screen is red
3. Tap / click immediately when it turns green
4. After 5 attempts, view your average reaction time
5. Tap to restart

---

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas API (for chart rendering)

No external libraries or frameworks are used.

---

## About This Project

This project was developed **with the assistance of AI** for code generation and structural guidance,  
while **all debugging, UX decisions, and platform-specific fixes were handled manually**.

Key human decisions included:
- Resolving mobile touch input conflicts
- Stabilizing layout across devices
- Handling platform limitations (Android vs iOS haptics)

# Project RT

**Project RT**는 사용자의 반응 속도를 측정하는  
미니멀한 웹 기반 reaction time 테스트입니다.

빨간 화면에서 대기하다가  
초록 화면이 나타나는 순간 탭 / 클릭하면  
반응 시간(ms)을 측정합니다.

---

## Features

- 🎯 Random delay 기반 반응 속도 테스트
- 🔁 5회 측정 후 평균 반응 시간 계산
- 📊 결과를 시각적으로 보여주는 그래프
- 📱 모바일 & 데스크톱 모두 지원
- ⚡ 매우 가벼운 단일 HTML 파일 구조

---

## Haptic Feedback (Mobile)

- **Android 기기**에서는 성공/실패 시 진동(햅틱 피드백)을 제공합니다.
- **iOS(iPhone)**에서는 웹 브라우저 정책상 진동 API가 제한되어  
  햅틱 피드백이 동작하지 않습니다.

> 이는 코드 문제가 아니라 운영체제 정책에 따른 정상 동작입니다.

---

## How to Use

1. 화면을 탭 / 클릭하여 시작
2. 빨간 화면에서 대기
3. 초록 화면이 나타나면 즉시 탭 / 클릭
4. 총 5회 측정 후 평균 결과 확인
5. 탭하여 다시 시작

---

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas API (for chart rendering)

외부 라이브러리 없이 순수 웹 기술만 사용했습니다.
